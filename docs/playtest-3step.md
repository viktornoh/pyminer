# 플레이 검증 절차 (3단계)

## 1) 상단 클리어 존/하강 체감 확인 (시작 10초)
- 실행: `./run.sh`
- 시작 직후 화면 최상단(지상 느낌 구간)에 블록이 비어 있는지 확인
- 조작 없이 잠시 관찰했을 때 과도한 자동 하강 느낌이 줄었는지 확인

## 2) 곡괭이 실루엣/크기 확인 (좌우 이동 20초)
- `A/D` 또는 `←/→` 로 좌우 이동
- 플레이어가 원형이 아닌 **명확한 곡괭이 실루엣(손잡이+헤드)** 으로 보이는지 확인
- 블록(40px) 대비 곡괭이의 체감 충돌/표시 면적이 약 2배 수준인지 확인

## 3) 타격 중심 플레이 확인 (1분 내)
- 블록 밀집 구간에서 연속 타격
- 단순 낙하보다 **이동+타격으로 파고드는 감각**이 강화되었는지 확인
- 위험 블록/일반 블록 충돌 시 피드백(밀림, 파티클, 점수/HP)이 정상 동작하는지 확인
